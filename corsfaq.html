1. What is a CORS proxy, and why is it needed?


Modern web browsers restrict websites from accessing data from another domain due to security rules called CORS (Cross-Origin Resource Sharing). A CORS proxy acts as a middleman that fetches the data on your behalf, bypassing those restrictions so your browser can display the MLB schedule data.



2. Is using a CORS proxy safe?


Yes! The CORS proxy we use is designed only to pass data through from the original source to your browser. It does not store, alter, or share your personal information. Think of it like a mail carrier delivering your package—it just moves data from point A to point B.



3. Why do I need to allow or enable the CORS proxy?


Since your browser blocks direct requests to external MLB data sources, allowing the CORS proxy is the only way we can fetch real-time schedule info directly within the website. Without it, you wouldn't be able to see the schedule data.



4. Will the CORS proxy slow down my browser or internet?


Not at all. The proxy simply routes your data and typically responds within milliseconds. It's optimized for speed and doesn’t run any heavy code in your browser.



5. Is my personal data exposed when using a CORS proxy?


No. The proxy only handles the public MLB schedule request. It does not access your private information, login details, or any browser history.



6. Can I disable the proxy if I want to?


Yes, but disabling it will prevent the MLB schedule data from loading. The proxy is only active while using this specific feature, and you’re always in control.
